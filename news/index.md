# Changelog

## primertools 0.2.0

- Added
  [`read_simper()`](https://anhsmith.github.io/primertools/reference/read_simper.md)
  to parse PRIMER one-way SIMPER text output files into structured
  within-group and between-group contribution tables.
- Added a vignette demonstrating how to read and work with PRIMER SIMPER
  outputs.

## primertools 0.1.1

- Add
  [`aic_permanova()`](https://anhsmith.github.io/primertools/reference/aic_permanova.md)
  and
  [`get_permanova_aic()`](https://anhsmith.github.io/primertools/reference/get_permanova_aic.md)
  for AIC-like comparison of PRIMER PERMANOVA models.
- Add vignette: “Reading PRIMER PERMANOVA outputs”.

### primertools 0.1.0

- Added read_permanova\_\*() helpers to parse PRIMER PERMANOVA output.
